3. Programozói nyelvek, fejlesztő eszközök

  • Frontend: HTML, CSS, JavaScript (esetleg React)
  • Backend: Node.js + Express (vagy más választott keretrendszer)
  • Adatbázis: MongoDB (vagy relációs: PostgreSQL)
  • Verziókezelés és együttműködés: Git + GitHub

4. Használni tervezett eszközök, technikák

  • Agilis munkamódszer (Scrum: rövid sprintek, heti standup)
  • Reszponzív dizájn — mobil és asztali nézet támogatása
  • RESTful API a frontend–backend kommunikációhoz
  • Alapvető hitelesítés (pl. JWT) és jogosultságkezelés

Várható produktumok: részletes projektjelentés (PDF), technikai dokumentáció, valamint egy bemutató (demo) verzió. A dokumentumok és a demo a félév végére, a záró prezentációhoz lesznek leadva.

  • Bejelentkezés / kijelentkezés stabil működése
